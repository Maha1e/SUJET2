# TP note sujet 2
***

Le tp porte sur des ordanceurs. L'objectif final est de recréer un ordanceur qui traierait des processus.

## Technologies
***

Liste des technologies utilisées:
* <stdio.h>
* <stdlib.h>
* <unistd.h>

## Utilisation
***

Il faut que les fichiers suivants se trouvent dans un même dossier:
* sujet2.c
* sujet2.h
* Makefile

Dans le terminal, se placer dans la direction correspondante au dossier du sujet2: 
```
make
./prog

```
